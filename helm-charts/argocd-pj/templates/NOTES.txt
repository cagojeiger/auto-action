1. Get the ArgoCD initial admin password by running:

   kubectl -n {{ .Release.Namespace }} get secret argocd-argo-cd-initial-admin-secret -o jsonpath="{.data.password}" | base64 -d

2. Access ArgoCD UI:

   {{- if index .Values "argo-cd" "server" "ingress" "enabled" }}
   ArgoCD will be available at: https://{{ index .Values "argo-cd" "server" "ingress" "hostname" }}
   {{- else }}
   kubectl port-forward service/{{ .Release.Name }}-argo-cd-server 8080:80 -n {{ .Release.Namespace }}
   Then open http://localhost:8080 in your browser
   {{- end }}

3. Login with:
   Username: admin
   Password: (from step 1)

4. ArgoCD Server Secret:
   The server secret key has been automatically generated and stored in the 'argocd-secret' Secret.
   This secret will persist across upgrades due to the 'keep' resource policy.
